Le Phénomène Open Data




Problématique


Aujourd'hui on produit plus de données que n'importe quel instant dans l'histoire des Data, un tas de données étant généré chaque jour.
La majorité de ces données restent stockée, peu accessible et pas totalement exploitée, alors que certaines informations peuvent offrir d"énormes opportunités en les reliant entre elles par exemple.

Une solution pour l'amélioration de l'exploitation des données serait de les ouvrir. D'où, l'apparition du Phénomène Open Data !

Outils de veille technologique




Qu'est ce qu'une veille technologique?




  • La veille est une activité continue et itérative faisant l'objet d'une surveillance active de l'environnement technologique, commercial... et acquérir ainsi de l'information à caractère stratégique ou opérationnel afin d'anticiper les évolutions de cet environnement, détecter les menaces et les opportunité pour l'amélioration du processus de décision.



  • La veille technologique ou veille scientifique et technique consiste à s'informer de façon systématique sur les techniques les plus récentes et aussi sur leur mise à disposition commerciale, ce qui constitue la différence entre la technique et la technologique. Pour l'entreprise, cette activité met en œuvre des techniques d'acquisition, de stockage et d'analyse d'informations et leur distribution automatique aux différentes sections concernées de l'entreprise, à la manière d'une revue de presse.

    • Open Data : Généralité


    Qu'est ce que l'Open Data

    ?


    L'Open Data: c'est l' ouverture au grand public des données collectées par:

  • des organismes publiques
  • des entreprises

    • et ce via le web ou les application mobiles, dont la diffusion est considérée comme d' intérêt général

    .

    Ces données regroupent généralement nos expériences quotidiennes telles que le transport, le déplacement, les conditions métérologiques, le parking ...

    Caractéristiques de l'Open Data



    Les données open data représentent certaines caractéristiques particulière. En effet, elles doivent être:


    de nature


  • complète

  • primaire

  • libre de droit (non soumises à des droits de propriété)

    • accessibles


  • publiquement

  • gratuitement

  • sans discriminations

    • au format


  • exploitable

  • non propriétaire (dont les spécifications sont publiques)

    • Formats privilégiés pour l'Open Data




    La démarche Open data privilégie les fichiers publiés dans des formats ouverts , c'est-à-dire qui ne sont pas dépendants d'un logiciel particulier et qui peuvent donc être lu avec n'importe quel logiciel.
    On distingue deux grands familles de formats habituels pour les données ouvertes, qui sont relatifs aux données tabulaires et aux données géospaciales.

    Les données tabulaires


    Pour les données tabulaire on retrouve les formats:


  • CSV : Comma Separated Value, format ouvert, il peut s'ouvrir dans n'importe quel logiciel tableur,

  • ODS : Open Document Spreadsheet, format ouvert du logiciel Calc de la suite bureautique libre et gratuite OpenOffice,

  • XLS : Excel, format fermé, propriétaire, de la suite bureautique Microsoft Office. Devenu un standard de fait, il peut s'ouvrir dans le logiciel Calc de la suite Open Office. Ce format est assez répandu malgrès le fait qu'il soit un format priviligié (n'est pas ouvert).


    • Les Données géographiques / géospatiales


    Pour les données géographiques (géospatiales) on retrouve les formats:


  • KML : Keyhole Markup Language, format ouvert, standard international utilisé notamment par les applications géographiques Google,

  • KMZ : format ouvert. C'est la version compressée du fichier KML, il contient les informations géo-référencées du fichier KML et en complément un dossier avec les images associées aux coordonnées,

  • geoJSON : Excel, : format basé sur JavaScript Object Notation – JSON,

  • SHP : Shapefile, format propriétaire, format ouvert, devenu un standard de fait et utilisé par de nombreux logiciels libres. 
    Les fichiers SHP se composent de quatre fichiers :

    1. le fichier .shp : contient les objets géographiques,
    2. le fichier .shx : index de la géométrie,
    3. le fichier .prj : contient les informations sur le système de coordonnées, il peut s'ouvrir dans un logiciel de traitement de texte ou un éditeur de texte,
    4. le fichier .dbf : contient les informations relatives aux objets géographiques, il peut s'ouvrir dans un tableur type Calc de Open Office.
